## CentOSGeoWebEnv

### Introduce

CentOSGeoWebEnv = OSGeo Web Environment for CentOS

* WebServer and Database

	* Nginx
	* MySQL
	* PostgreSQL
	* Sqlite

* OSGeo

	* Proj4
	* GEOS
	* Spatialite
	* GDAL
	* PostGIS
	* Mapnik
	
* Interprater

	* Python
	* NodeJS(todo)

	sudo su root
	./install.sh
