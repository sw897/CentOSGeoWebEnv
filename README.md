## CentOSGeoWebEnv

### Introduce

CentOSGeoWebEnv = OSGeo Web Environment for CentOS

* WebServer

	* Nginx

* Database

	* MySQL
	* PostgreSQL/PostGIS
	* Sqlite/Spatialite(not recommended)

* OSGeo

	* Proj4
	* GEOS
	* GDAL
	* Mapnik

* Interprater

	* Python2.7
	* NodeJS(todo)


### Install

	sudo su root
	./install.sh
